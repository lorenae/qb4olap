QB4OLAP

A RDF vocabulary to publish and operate with multidimensional data on the web.

Download this project as a .zip file Download this project as a tar.gz file

Introduction

QB4OLAP vocabulary is an extension to the DataCube vocabulary that allows to represent OLAP cubes in RDF, and to implement OLAP operators (such as Roll-up, Slice, and Dice) as SPARQL queries directly on this RDF representation.

You can find the current version of the vocabulary in Turtle format here.
